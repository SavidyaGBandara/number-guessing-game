# Number Guessing Game

A simple command-line number guessing game built with Python. This is one of my early Python learning projects.

## How to Play

1. Run the script
2. Choose a difficulty mode
3. Guess the number — the game will tell you if you're too high or too low
4. Try to guess it in as few attempts as possible!

## Difficulty Modes

| Mode   | Range  |
|--------|--------|
| Easy   | 1 – 20 |
| Medium | 1 – 40 |
| Hard   | 1 – 80 |

## Features

- Three difficulty modes
- Proximity-based hints (e.g. *"High but close"*, *"Too Low"*)
- Attempt counter
- Option to give up and reveal the answer
- Play again prompt

## Requirements

- Python 3.x
- No external libraries needed
